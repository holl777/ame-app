# ÂME — Bien-être & Élévation

Application web progressive (PWA) de bien-être, spiritualité et croissance personnelle.

## Installer l'app

### iPhone / iPad
1. Ouvrir Safari et aller sur l'URL de l'app
2. Appuyer sur le bouton **Partager** (carré avec flèche)
3. Défiler et choisir **"Sur l'écran d'accueil"**
4. Appuyer sur **Ajouter**

### Android
1. Ouvrir Chrome et aller sur l'URL de l'app
2. Une bannière "Installer ÂME" apparaît automatiquement
3. Appuyer sur **Installer**
4. Ou : menu ⋮ → "Ajouter à l'écran d'accueil"

## Déploiement GitHub Pages

1. Uploader tous ces fichiers sur GitHub
2. Settings → Pages → Source: main branch
3. L'app sera disponible à `https://[username].github.io/[repo-name]`
